This is a landing page in connection to my portfolio project "A maze game". The maze is a 3D maze game that uses ray casting to render a 2D map into a 3D navigatable world.

Usage:

  • Execute ./maze or type make run
  • Use up and down arrow keys to move forward and backward (keys W and S serve the same function)
  • Use right and left arrow keys to turn the camera around (Keys D and A serve the same function)
